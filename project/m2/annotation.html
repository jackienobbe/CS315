Usage of PHP

  • For this milestone, PHP was primarily implemented submit bands and shows. Because there is a heavy reliance on a database, the website now just takes the form data and displays it on the appropriate page as though it were pulling from a database.
    • Adding a new band forwards to viewBand.php, displaying the recent input.
    • Adding a new show forwards to viewShow.php, displaying the recent input.
  • The start of a login system is included, but does not affect much at this time. Simply takes username "aquadome" and password "password".
